Estrutura básica do HTML

O HTML é uma linguagem de marcação onde será exibida a informação e serve para dar significado e organizar/estruturar as informações nos sites. HTML é abreviação em inglês de HiperText Markup Language e consiste em basicamente 3 partes:

  1. Tag HTML
  2. Tag head
  3. Tag body

A tag HTML representa a raíz de um documento html e é o recipiente para todos os outros elementos.

O conteúdo dentro da tag head contém informações que não são transpostas visivelmente para o usuário.

Tudo que estiver dentro da tag body é o que irá aparecer para o usuário. O documento em si.

Falando sobre tags

Tags são o conjunto de caracteres que formam um elemento. Existem dois tipos de Tags, as que necessitam de fechamento e as que não necessitam de fechamento.

Listas

Existem duas tags para serem usadas quando precisarmos adicionar listas na nossas páginas.

Temos a Lista Ordenada (onde usamos a tag ol) e entre essa tag usamos a tag li para montar a lista.
Como o nome já diz, essa tag nos dá uma lista em ordem númerica.

O outro tipo de lista que temos é a Lista não ordenada. Para colocarmos essa lista em nosso site utilizamos a tag ul. Entre essa tag usamos a tag li para montar a lista. Essa tag nos dá uma lista fora de ordem e com marcações.

Links

O link HTML é uma função do HTML que usamos para inserir hiperlinks em alguns elementos e é necessário que esteja apontando para um endreço existente. Existe um atributo, que até aqui onde estou nesse momento nos estudos, que parece ser o ou uns dos principais dessa tag.

Esse atributo é o href="". Através desse atributo podemos redirecionar o usuário a outra página do documento ou a outro recurso.

Podemos usar 3 tipos de links dentro do href e são eles interno, local e global.
